# STL提出確認

**STL提出確認** は、48mmガチャポンFITで生徒がサイズ調整を行い、決められたルールで保存したSTLファイルを、クラスごとに一覧表示して確認するためのWebアプリです。
共有フォルダなどで集めた提出STLをZIPファイルとして読み込み、ファイル名に含まれる学年・クラス・番号・提出日時・氏名情報をもとに、提出状況と立体データを確認できます。

公開ページ：  
https://jkawamata.github.io/stl_file_list/

---

## 主な用途

このアプリは、授業で生徒が作成したSTLファイルを、教師が効率よく確認するためのものです。
特に、次のような場面を想定しています。

- 48mmガチャポンFITで保存されたSTLファイルを一覧確認する
- クラスごとの提出状況を確認する
- 生徒ごとの提出回数を確認する
- 提出されたSTLの形状をサムネイルで確認する
- STLを個別に拡大表示して確認する
- クラスごとのSTLサムネイル一覧を印刷する
- 読み取れなかったSTLファイルを確認する

## 使い方

### 1. 提出されたSTLファイルをZIPで用意する

共有フォルダで集めた48mmガチャポンFITのSTLファイルを、提出されたフォルダーごとZIPファイルとしてダウンロードします。
共有フォルダ内にさらにフォルダがある場合でも、ZIPファイルの中に含まれていれば読み込めます。

### 2. ZIPファイルを読み込む

アプリを開いたら、次のどちらかの方法でZIPファイルを読み込みます。

- ZIPファイルを画面にドラッグ＆ドロップする
- 「ZIP選択」ボタンからZIPファイルを選ぶ

ZIPファイルを読み込むと、ZIP内のSTLファイルを解析し、提出一覧を作成します。

### 3. 学年・クラスを選ぶ

読み込み後、学年とクラスを選択します。

- 学年は1年〜9年から選択できます
- クラスは1組〜9組から選択できます

10組は選択できない仕様です。

### 4. 番号の範囲を選ぶ

生徒番号の表示範囲を選択できます。

標準では40番まで表示します。必要に応じて、次の範囲に変更できます。

- 10番まで
- 20番まで
- 30番まで
- 40番まで
- 50番まで
- 60番まで
- 70番まで
- 80番まで
- 99番まで

通常の中学校のクラスでは40番までで十分ですが、講座・学年全体・特別なグループなどでも使えるようにしています。

## 対応するファイル名ルール

このアプリは、48mmガチャポンFITで保存されたSTLファイル名をもとに情報を読み取ります。

基本的なファイル名の例は次の通りです。

```text
1407_20260624_1515_山田太郎.stl
```

この場合、次のように読み取ります。

| 部分 | 意味 |
|---|---|
| `1` | 1年 |
| `4` | 4組 |
| `07` | 7番 |
| `20260624` | 2026年6月24日 |
| `1515` | 15時15分 |
| `山田太郎` | 氏名 |

つまり、先頭の番号コードは次のような意味になります。

```text
1101 → 1年1組1番
1407 → 1年4組7番
1909 → 1年9組9番
```

クラスは1組〜9組を想定しています。  
番号は最大99番まで扱えます。

## 表示モード

### サムネイル一覧

クラス全体の提出STLを一覧で表示します。

各生徒の行には、次の情報が表示されます。

- 番号
- 氏名または匿名名
- 提出個数
- STLサムネイル
- 提出回数
- 提出日時

未提出の生徒は「未提出」と表示されます。

### 個別表示

生徒を選択すると、その生徒が提出したSTLを個別に確認できます。

複数回提出している場合は、提出回数ごとにサムネイルが表示されます。

STLサムネイルは、ドラッグして回転できます。  
ホイール操作で拡大・縮小できます。  
ダブルクリックすると、より大きく表示できます。

---

## STL表示機能

STLファイルは、ブラウザ上で3D表示されます。

主な特徴は次の通りです。

- STLファイルをサムネイル表示できる
- 等角図に近い向きで表示できる
- 個別に回転して確認できる
- 拡大・縮小して確認できる
- 複数回提出されたSTLを並べて比較できる
- 印刷用の静止サムネイルを生成できる

STLの表示には外部ライブラリを使用していますが、読み込んだSTLファイルの情報はブラウザ上で処理されます。

---

## 印刷機能

「サムネイル一覧」は印刷できます。

印刷時には、クラスごとの提出STLが一覧形式で出力されます。  
提出状況の確認、評価時のメモ、授業後の記録などに利用できます。

印刷画面では、不要な操作ボタンやステータス表示は出ないように調整しています。

---

## 読み取れなかったSTLの確認

ファイル名のルールに合わないSTLや、情報を正しく読み取れなかったSTLは、通常のクラス一覧とは別に扱われます。

「読み取れなかったSTL」がある場合は、その個数を確認できます。

読み取れなかったSTLについても、サムネイルで形状を確認できます。

---

## 個人情報と通信について

STLの表示のために外部ライブラリを使用していますが、ZIPファイルで読み込んだSTLファイルの情報は、このアプリが動くブラウザ上で処理されます。

生徒の個人情報がネット上に送信されることはありません。

ただし、アプリ本体や外部ライブラリを読み込むためには、ページ表示時にインターネット接続が必要です。  
ZIPファイルやSTLファイルの中身を外部サーバーにアップロードする仕組みはありません。

---

## 注意事項

- ファイル名が決められたルールと異なる場合、正しく一覧に反映されないことがあります
- STLファイルが非常に多い場合、表示に時間がかかることがあります
- 端末の性能によっては、大量のSTLサムネイル表示が重くなる場合があります
- iPadやiPhoneでは、PCと比べてファイル操作や表示が異なる場合があります
- 印刷結果はブラウザやプリンタ設定によって変わることがあります
